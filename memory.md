## Usage

Update this file after each chapter completion:
1. Add new character entries as they appear
2. Update timeline with events
3. Add or modify world rules as established
4. Track plot point progression
5. Note thematic element appearances
6. Log chapter status in progress table

All four agents read and write to this file to maintain story consistency:
- **Architect**: Initializes structure, characters, world, themes, market notes
- **Writer**: Updates discovered details during drafting
- **Dev Editor**: Logs structural decisions and revision notes
- **Line Editor**: Records style decisions and compliance flags
